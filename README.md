# embedded-linux-utils

Modern C++20 utility library for Embedded Linux applications.

## Overview

**embedded-linux-utils** is a collection of reusable C++ utilities designed for Linux-based embedded systems and server applications.

The project focuses on clean architecture, modern C++ practices and production-ready components that simplify the development of reliable Linux software.

## Features

* Logging framework
* Configuration parser
* File system utilities
* Process management
* Network utilities
* String and time utilities
* Thread-safe components
* Modern CMake build system
* Unit tests
* GitHub Actions (CI)
